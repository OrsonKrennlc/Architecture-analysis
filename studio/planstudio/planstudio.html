总平上色 · 本地算法 · 无 AI
  1. 上传总平图(线稿或渲染图都行)
  2. 想快出风格 → 顶部风格条点一个(可选)
  3. 左栏分两步:先选「怎么选」(自动点填 / 多边形框选 / 画笔涂),再选「填什么」材质 → 点图上区域
  4. 填不满?把选区灵敏度往左拉;碎了点一键清理碎块
  5. 拖太阳定光影 · 拖动种树 / 配景 · 逐栋楼高点楼让高楼影子更长
  6. 加标注(指北针 / 比例尺 / 标定真实比例 / 尺寸 / 标高 / 图框标题)
  7. 右上导出 PNG / JPG / PDF,可选分辨率与透明底
边界有缺口、漏色填到整张图 → 那段边没闭合,改用多边形框选最稳。
